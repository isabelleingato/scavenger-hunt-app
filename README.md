# Scavenger Hunt App

Web-search based "scavenger hunt" game composed of a client and server application, along with a Chrome extension.

## Running Locally?

1. `brew install mongodb-community@4.4`
2. `mkdir mongodata`
3. `mongod --dbpath mongodata`
